    • Project One: Personal Loan Defaul

      Extracted insights regarding default rates based on customer features such as; loan amounts, purpose, age, and homeownership implications. Developed visualizations for data representation while exploring ethical loan decision-making processes.

    • Project Two: Diabetes type 2 Onset predictors

      Applied Machine Learning Algorithms to predict the features that precede the onset of Diabetes Type 2 along with visualizations. Used the Accuracy Score to test my model and explored strategies to improve prevention and early detection.

    • Project Three: Linear Regression in Health Insurance costs

      Applied Linear Regression Models to predict features that contribute to the cost of Health Insurance on a dataset. Performed three different experiments with the data to find the best features for the regression model.

    • Project Four: Social Media and Mental Health

      Applied Clustering Algorithms to discover patterns and trends to help us gain insights about the effects of social media and mental health. Applied the "elbow method" to propose the optimal number of clusters and implemented the "Principal Component Analysis" to reduce the dimensionality of data to improve the clustering algorithm's performance.

    • Project Five: Recommendation System

      Implemented a model that uses a User-based filtering approach to provide a reliable and robust model for a recommendation system for two merged data sets containing movie ratings and titles
