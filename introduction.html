Skills and Experiences with data

Welcome to my personal project section. As a rising senior, I chose Data Science as my primary area of concentration. My fascination has always revolved around data and its potent applications within machine learning. The projects I've worked on include data pre-processing techniques and machine learning algorithms. Through these projects, I have developed a set of marketable Data Science skills. Some of these skills include Python for data analysis and machine learning, data processing libraries like Pandas and Numpy, knowledge in data visualization using tools such as Matplotlib and Seaborn, and hands-on experience using machine learning models with Scikit-learn.

    • Project One: Personal Loan Defaul

      Explored a Kaggle dataset to find determinants of loan defaults. Performed data preprocessing by addressing null values and refining key columns. Extracted insights regarding default rates based on loan amounts, purpose, age, and homeownership implications. Presented data visualizations to enhance personal loan risk assessment strategies while promoting ethical loan decision-making processes.

    • Project Two: Diabetes type 2 Onset predictors

      Analyzed a Kaggle dataset to identify key predictors of Diabetes Type 2. Performed data preprocessing, addressing outliers and refining essential variables. Highlighted the impact of factors such as age and blood glucose levels on disease onset. Used visualizations to clearly present these correlations, aiming to improve prevention and early detection. Applied Machine Learning Algorithm (Random Forest) and used Accuracy Score to identify the features more likely to predict the onset of Diabetes Type 2.
